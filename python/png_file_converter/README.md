# 文件-图片转换器
> 将文件转换为图片的工具

## 说明
- 将任意文件转化为png图片存储
- 目前文件大小需要在大约5M以内
- 文件大小后续可以优化(比如多张图片或者更大的图片什么的)

## 用法
- 直接调用Python文件
- 可以用测试文件试一下加解密
- 讲道理对任何文件应该都可以进行这个操作
- 参数的调用方法以后如果有空的话再加上来

## 算法
> 其实没什么复杂的……

1. 将文件以二进制的方式读取
2. 将读取的字节三个一组转为颜色信息
3. 颜色信息再转储到图片的像素点上
